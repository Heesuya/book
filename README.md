# 도서관리를 위한 웹 프로그램 


## 기술 스택

- **운영체제**: macOS
- **사용언어**: Java, JavaScript, HTML5, CSS3
- **사용기술**: Spring Framework, React, REST API, MyBatis
- **Framework/Library**: Node.js, Spring Framework
- **DB**: Oracle 11g
- **Tool**: Visual Studio Code, STS, SQL Developer
- **Collaboration**: GitHub, ERD
